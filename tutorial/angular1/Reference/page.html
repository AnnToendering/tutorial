``` ## The `ons` object Onsen UI not only provides custom elements, it also provides an object called `ons` with a lot of useful functions attached to it. In this example we set a click handler to `ons-button` with `ng-click` attribute. This button is inside the previously declared `PageController` scope. ## Displaying a dialog Onsen UI also provides functions for displaying dialogs. In this example we are using `ons.notification.alert` to show an alert dialog. There are also `confirm` and `prompt` methods available to show other types of dialogs. You can try replacing the click handler with the following code to get another type of dialog: ``` ons.notification.confirm({message: 'Are you ready?'}); ``` We will take a look at other types of dialogs in a later section. -->
